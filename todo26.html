구글 시트 백업

Google Apps Script 웹앱 URL을 연결하면, 버튼을 눌러 변경된 항목만 시트에 업로드하거나 시트 내용을 불러올 수 있어요. 자동 동기화는 아니며, 항상 버튼을 눌러야 동작합니다. 설정 방법은 아래 링크 참고.

설정 방법 보기 (최초 1회, 5분)
구글 시트 연결 방법
  1. 새 구글 시트를 만듭니다 (sheets.google.com → 빈 스프레드시트).
  2. 상단 메뉴 확장 프로그램 → Apps Script를 클릭합니다.
  3. 기본 코드를 모두 지우고, 아래 "코드 복사" 버튼으로 받은 코드를 붙여넣습니다.
  4. 저장(💾) 후, 우측 상단 배포 → 새 배포를 클릭합니다.
  5. 유형 선택에서 웹 앱을 고르고, "액세스 권한"을 모든 사용자로 설정 후 배포합니다.
  6. 발급된 웹앱 URL(.../exec 로 끝남)을 복사해서, 위 입력칸에 붙여넣고 저장합니다.
